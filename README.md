## 🎯 Rules
* 그날 배운 내용은 미루지 않고 **당일에 복습**합니다. 🔥
* 단순히 복사/붙여넣기가 아닌, **내 언어로 소화한 내용**만 기록합니다.
* 코드는 **Clean Code**를 유지하려고 노력합니다.

<br/>

## 💬 Commit Convention
> 한눈에 알아보기 쉽게 이모지를 사용합니다.

| 이모지 | 태그 | 설명 |
|:---:|:---:|:---|
| ✨ | Feat | 새로운 기능 추가 |
| 🛠️ | Fix | 버그 수정 |
| 📝 | Docs | 문서 수정 (README, TIL 등) |
| 🎨 | Style | 코드 포맷팅, 세미콜론 누락 등 (로직 변경 X) |
| ♻️ | Refactor | 코드 리팩토링 |
| ✅ | Test | 테스트 코드 추가/수정 |
| 🚀 | Deploy | 배포 관련 |
